Todo List(Lista de Tarefas)

O projeto foi desenvolvido para colocar em prática os conehecimento de HTML, CSS e Javascript adquiridos nos cursos da B7W. Nele foi possível colocar em prática a manipulação dos elementos HTML utilizando Javascript. O site utiliza o recurso do localStorage para fazer o armazenamento das tarefas no navegador do cliente, sendo assim, após o navegador ser fechado, ou até mesmo o computador reiniciado, as tarefas continuam aparecendo do mesmo jeito que foi deixado desde a última interação do cliente. Foi utilizado os serviços da Vercel para fazer o deploy e hospedagem do site.
