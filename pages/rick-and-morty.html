API Rick and Morty

O projeto da API do rick and Morty foi criado para colocar em prática os conhecimentos adquiridas na matéria de Programação Orientada á Obetos Avançada no curso de ADS da Fatec. Nele foi utilizado a biblioteca do axios para tratar requisições HTTP e async await para as promises. O projeto foi hospedado utilizando os serviços do GitHub Pages.
